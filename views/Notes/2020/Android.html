        Yang Cui 2020-4-24 Android Notes

        # Ionic

        # Preparation

        1. Install npm https://nodejs.org/en/
        2. Install JDK and Configure JDK environment variables https://www.oracle.com/java/technologies/javase-jdk8-downloads.html
        3. Install Android Studio https://developer.android.com/studio
        4. Install cordova, ionic by npm

        # Create Ionic Project

        ionic start <project name> tabs
cd <project name>
ionic serve

        # Package

        # View supported platforms or versions
ionic cordova platform ls
# Add Android Environment to the Project
ionic cordova platform add android
# if have some problems, remove and add again
ionic cordova platform remove/rm android
# import project to Android Studio to run, project-platforms-android or build.gradle
ionic build [--prod]
ionic cordova prepare <platform>
# mean to work build,prepare,run app on the android phone
ionic cordova run android
# view plugins
cordova plugin list

        use 'ionic cordova run android' need to deploy environment variables

        GRADLE_HOME

        ANDROID_HOME

        ANDROID_SDK_ROOT

        in 'Path', add below infos,

        %GRADLE_HOME%\bin; %ANDROID_SDK_ROOT%\emulator;%ANDROID_SDK_ROOT%\build-tools

        From the above, the app can be test/debug in the Android phone.

        Release versions need below steps,

        In Android Studio,

        1. Build
2. Generate Signed Bundle/APK
3. Android App Bundle(Android 9.x above)/APK
4. Fill the forms, then release the .apk file to AppStore
5. After update the app, change versionCode and versionName in AndroidManifest.xml, then repeat step 1-4
