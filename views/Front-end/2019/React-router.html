    # location

    • pathname：当前地址
    • query：传参可以string or object, 地址？id=xxx
    • state：传参，一般用于传递地址 from-to，匿名变量传参，传递来源 给你钩子
    • key
    • search：用new URLSearchParams(location.search)
    • hash

    # match

    • params：url传参 /：id
    • isExact：bool，它是当前location与link是否严格匹配
    • path：匹配到表达式（eg. path='/love/:id'）
    • url：匹配到路径后面具体那部分（eg. path='/love/1111'）

    # withRouter(): 不能传参，只能帮助拿到内置参数，进行后续操作，它是一个高阶组件，需要在react-router-dom导入它
